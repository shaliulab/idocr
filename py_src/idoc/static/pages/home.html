Welcome to the IDOC Platform @ Liu Lab

Welcome to the Individual Drosophila Optogenetics Conditioner. This program allows you to schedule hardware events with an ArduinoMega board so you can interact with flies and condition their preference for stimuli. The software comes with a Tracker module that enables quantification of the pre and post-conditioning preference. Conditioning can be performed by exposition to either an a priori neutral odor or an optogenetically active light, paired with an electro shock. The software allows you to fine tune the intensity of the electro shocks and the intensity of the optogenetic light. Analysis is done with R.
